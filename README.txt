# PETSTORE KARATE TEST

## Herramientas
- JDK 17
- Apache Maven 3.9+
- Karate 1.5.1
- IntelliJ IDEA o VS Code

## Pasos de ejecución
1. Clonar o descomprimir el proyecto.
2. Abrir una terminal en la carpeta raíz del proyecto.
3. Ejecutar:
   mvn test

## Orden sugerido de ejecución
1. AddPet.feature
2. GetPetById.feature
3. UpdatePet.feature
4. GetPetByStatus.feature

Los reportes HTML se generan automáticamente en:
target/karate-reports/karate-summary.html
